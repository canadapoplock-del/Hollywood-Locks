Vercel is excellent for hosting static sites like this one.

Upload to GitHub

Create a new repository on GitHub.

Upload index.html, styles.css, and script.js to the repository.

Connect to Vercel

Log in to Vercel.

Click "Add New..." -> "Project".

Import your GitHub repository.

Configure

Framework Preset: Select "Other" (since this is a plain HTML site).

Root Directory: Leave as ./.

Build Command: Leave empty.

Output Directory: Leave empty.

Deploy

Click "Deploy".

Vercel will build your site and give you a live URL (e.g., hollywood-locks.vercel.app) in seconds.
